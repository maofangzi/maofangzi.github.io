这是一个关于hugo搭建博客的介绍

hugo

Hugo是一个用Go编写的静态站点生成器,非常好用（fake）。

hugo安装

window手动安装

下载的hugo.exe配置全局环境变量

hugo version

快速安装

macOS使用Homebrew

brew install hugo

Windows使用Chocolatey

choco install hugo -confirm

linux使用apt

sudo apt-get install hugo

1、创建静态站点

hugo new site myblog

2、添加模板

cd myblog
cd themes
git clone https://github.com/NightFarmer/hugo-theme-yelee.git

3、配置默认模板

myblog 目录下 config.toml 修改

baseurl = "http://bolg.aliveyang.top/"
languageCode = "en-us"
title = "Fang Yang"
theme = "hugo-theme-yelee"

4、添加文章

该模板文章对应 /myblog/content/post 下面md文件 模板头信息添加

+++
## Categories = ["Development", "GoLang"]
Description = ""
Tags = ["test"]
date = "2018-09-23T16:30:37+08:00"
menu = "main"
title = "你好，FY"
+++

5、本地运行

myblog目录下

正常运行
hugo server
调试模式（data,content,layouts,static,themes/xxx）
hugo server -w

访问 localhost:1313

6、生成静态html

myblog目录下

hugo -D

生成静态文件在/myblog/public中，上传至github page即可
